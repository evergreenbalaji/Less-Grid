About

A simple & basic grid system for web pages using LESS CSS pre-processor.
It is a striped-down simple version of the grid systems inspired by Twitter Bootstrap, Skeleton & other frameworks.

Download View on Github

Usage

CSS version has simple & self-evident class names to reduce complexity from the other grid systems.
The LESS version is primarily a mixin, so the grids can implemented flexibily, Check out the demo.

CSS

<!--HTML-->
<div class="row">
   	<div class="Six"> </div>
   	<div class="Six"> </div>
</div>

/* CSS */
.Six {
	min-height: 1px;
	padding-left: 15px;
	padding-right: 15px;
	box-sizing: border-box;
	background: #ED5565;
}
@media (min-width: 769px) {
	.Six {
    	float: left;
    	width: 50%;
  	}
}

LESS

<!--HTML-->
<div class="row">
   	<div class="Six"> </div>
   	<div class="Six"> </div>
</div>

/* LESS */
.row { 
  .grid-row(); 
}
.Six{
  .grid-column(6);
  background:#ED5565;
}
